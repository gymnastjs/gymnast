Named spacing sizes offer an easy way to set the padding or margin for an element.

| Valid Values | Pixel Size |
| ------------ | ---------- |
| XS           | 4px        |
| S            | 8px        |
| M/2          | 16px       |
| M            | 12px       |
| L/2          | 24px       |
| XL/2         | 16px       |
| XL           | 32px       |
| XXL/2        | 24px       |
| XXL          | 48px       |
